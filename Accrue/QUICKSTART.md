### 完整操作流程（3分钟）

#### 1️⃣ 连接钱包
```
✓ 点击右上角"连接钱包"
✓ MetaMask 弹出 → 点击"下一步" → "连接"
✓ 看到钱包地址显示 = 成功！
```

#### 2️⃣ 锁定仓位
```
✓ 选择金库（BTC $100k 看涨 或 看跌）
✓ 输入数量（比如 100）或点击"最大"
✓ 点击"授权代币" → MetaMask 确认
✓ 等待交易确认（约 5-10 秒）
✓ 点击"锁定仓位" → MetaMask 确认 3 次
✓ 看到"仓位锁定成功" = 完成！
```

#### 3️⃣ 借入 USDC
```
✓ 在"释放流动性"页面
✓ 选择"自主管理"
✓ 输入借款金额或点击"最大"
✓ 点击"借入 USDC" → MetaMask 确认
✓ 看到"USDC 借入成功" = 完成！
```

#### 4️⃣ 查看收益
```
✓ 自动跳转到"收益管理"页面
✓ 查看健康因子（应该 > 1.5）
✓ 观察"累计收益"每秒增长
✓ 🎉 您正在赚取被动收益！
```

**Q: 我看不到余额？**
```
A: 确认：
1. 钱包已连接
2. 网络是 Polygon Amoy
3. 已获取测试代币
4. 刷新页面
```

**Q: 交易失败了？**
```
A: 检查：
1. MATIC 余额是否充足（用于 gas）
2. 代币余额是否充足
3. 是否已授权

```
A: 两个选择：

A: 两个步骤：
1. 先偿还债务
2. 再取回抵押品
